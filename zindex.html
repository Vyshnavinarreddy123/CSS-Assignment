Z-Index Demonstration

Overlapping elements with different z-index values:

Box 1 (z-index: 1)
Box 2 (z-index: 2)
Box 3 (z-index: 3)
Box 4 (z-index: 4)

Notice how the elements overlap and are stacked based on their z-index values.

Now, let's change the positioning methods:

Box 1 (z-index: 1, absolute)
Box 2 (z-index: 2, fixed)
Box 3 (z-index: 3, relative)

See how the positioning methods interact with the z-index values to affect the stacking order.
