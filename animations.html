CSS Animations Demonstration

Animations:

Spin animation (2s, linear, 1s delay, infinite iterations)
Bounce animation (1s, ease-in-out, 2s delay, 3 iterations)
Fade animation (3s, ease, 0s delay, 1 iteration)

Notice how the animations are applied to the elements using the animation-name property.

Experiment with different animation properties like animation-duration, animation-timing-function, animation-delay, and animation-iteration-count to achieve different effects.
